Mikroserwisy REST API

Rest API na bazie mikroserwisów. Projekt, który zajął mi zdecydowanie najwięcej czasu na implementację. Ma przemyślaną logikę biznesową, która pozwala na rejestr studentów, rejestr kursów a także możliwość zapisu studenta na dany kurs. Wszystko w oparciu o spring boot.
Pierwotnie korzystałem z bazy MongoDB i Postgresa jednak niedawno edytowałem projekt zostając tylko przy Postgresie. Do weryfikacji projektu używałem Postmana. Oczywiście napisałem też komplet testów jednostkowych przy użyciu JUnit i Mockito a także Jacoco do sprawdzenia pokrycia testów w kodzie. Apka posiada Gateway który rozdziela zapytania zależnie od potrzeby, nie trzeba wysyłać zapytań bezpośrednio do danego serwisu.
Następnym krokiem rozbudowy aplikacji jest stworzenie frontendu, planuję aby zrobić to w oparciu o React. Przedstawiam krótką galerię z Apki oraz zachęcam do zapoznania sie z jej kodem na github (link pod galerią).
